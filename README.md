# 🎵 Spotify Clone

A Spotify Clone web application built using HTML, CSS, and JavaScript.

## 🚀 Features

* Play / Pause songs
* Next / Previous controls
* Progress bar
* Interactive UI

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📚 What I Learned

* DOM Manipulation
* Audio handling in JavaScript
* Event handling
